<demo-demo>

A somewhat simple way to create a step in a tutorial with HTML, CSS, and Javascript visible.

Example

I found this component useful for creating tutorials where a other code (usually other web components!) is explained in a step-by-step fashion.

Produced from this code:

      <demo-demo class=example id="example-1">

      <figure>
        <span class="person">🧍</span>
        <span class="castle">🏰</span>
      </figure>

      <style>
      #example-1 span {font-size:4rem;}
      #example-1 figure { display: flex;}
      </style>

      <script  type="module">
      document.querySelector('#example-1 .person')
      .addEventListener("click", e => 
        e.target.style.transform = 'scale(2)'
      )
          </script>

      </demo-demo>

Usage

This component is opinionated. You must have the following children in your markup, in this order:

  1. <figure></figure>
  2. <style></style>
  3. <script type="module"></script>

The <pre> tag containing the markup inside the <figure> tag is automatically generated.

The <style> tag may also have a contenteditable attribute to make the CSS interactively editable. This is not possible for the <script> tag, as far as I know.

Note also that the rules in the <style> tag are scoped to the component itself. This would probably be better with shadow DOM and :root{} and all that, but I generally don’t like it, and I don’t find this scoping rule onerous.

In all likelihood you will also want to scope selectors in the <script> tag to the component as well, as I have done here.
