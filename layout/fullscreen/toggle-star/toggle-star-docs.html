<toggle-star>

A component for a toggleable star.

Example

<toggle-star></toggle-star>

This very simple component provides a clickable star for rankings, tagging things as important, etc.

Attributes

starred

The default is for the star to be unselected. You can change that by adding a starred attribute. 

<toggle-star starred></toggle-star>
